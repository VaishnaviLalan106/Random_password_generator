# Random Password Generator

This is a small project I created while learning programming.  
I tried implementing the logic of generating random passwords in **Python** and also recreated a similar idea using **HTML, CSS, and JavaScript**.

The goal of this project was simply to practice programming concepts and understand how random password generation works.

---

## What this project contains

1. **Python Version**
   - Generates a random password in the terminal
   - Uses Python's `random` and `string` modules

2. **Web Version**
   - Built using HTML, CSS, and JavaScript
   - Allows users to:
     - Choose password length
     - Include uppercase letters
     - Include lowercase letters
     - Include numbers
     - Include symbols
   - Shows password strength and allows copying the password
